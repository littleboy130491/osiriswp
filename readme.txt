A lightweight starter plugin scaffold for custom site enhancements. Adds a settings page, basic assets, and a demo shortcode.

== Description ==
- Settings: Settings → OsirisWP to configure a greeting.
- Shortcode: [osiriswp_hello name="world"] outputs greeting + saved option.

== Installation ==
1. Upload or place this folder in `wp-content/plugins/osiriswp`.
2. Activate via Plugins → Installed Plugins.

== Frequently Asked Questions ==
= How do I change the greeting? =
Visit Settings → OsirisWP and update the field.
